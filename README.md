# NanoCem UT-9 — Product Website

Premium landing page for **NanoCem UT-9** — ultrafine grouting cement by EasyMix / ТОО ЭкоМикс.

## Features
- 🎲 Interactive 3D product bag (mouse tracking)
- 🌐 Animated particle background network
- 📊 Live Chart.js chart with real well data (Well 343, Karatyube)
- 🎮 Interactive canvas demo — NanoCem vs Portland cement penetration simulation
- 📏 Particle size comparison bars
- 🏁 Phase timeline with key KPIs
- 📋 Full technical specifications table

## Tech Stack
- Vanilla HTML5 / CSS3 / JavaScript
- Chart.js 4.4.0 (CDN)
